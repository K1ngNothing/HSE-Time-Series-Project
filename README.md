Структура репозитория:
* `report.md`: отчет по эксперименту. Формально описывает проверяемую гипотезу, ход эксперимента, полученные результаты. По сути мини-статья по эксперименту
* `project.ipynb`: содержательный код экспериментов. Поэтапно выполняет эксперимент, генерирует вспомогательные графики.
* `/src`: набор технический скриптов, используемых в `project.ipynb`. Туда вынесена работа с датасетами, создание фичей, и т.п.
* `config.py`: константы, используемые в эксперименте (например, `SEED`)
